# Wazuh-SIEM-Active-Directory-Labs
# Overview
The lab documents the integration of Wazuh SIEM with Windows Server 2022 (Active Directory Domain Controller) and Windows 11 Clients to collect and analyze security-relevant events. The focus area include
- Authentication and Logon activity
- Group Policy and Policy changes
- Account and group management
- Core Windows security events
The goal is to simulate a real SOC-style setup, validate log collection, map detections to MITRE Att&amp;ck and document finding using incident style reports.


# Skills Demonstrated
- SIEM deploymenet and agent management (Wazuh)
- Windows Security auditing and GPO configuration
- Active Directory monitoring and log analysis
- MITRE ATT&CK mapping
- Incident style Documentation and remediation analysis
